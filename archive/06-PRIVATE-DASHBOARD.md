## Философия Dashboard

Dashboard — это не "список видео". Это **домашний экран** пользователя в продукте.

Три правила:
1. **Ясность** — с первого взгляда понятно, где ты и что делать
2. **Мотивация** — прогресс виден, следующий шаг очевиден
3. **Возвращаемость** — удобно продолжить после перерыва

## Главный экран — три состояния

### Состояние 1: Первый визит

```
Привет, [Имя]! Добро пожаловать.

Курс состоит из 5 уроков. Каждый строится на предыдущем.
Начни с первого — он займёт ~15 минут.

[Начать первый урок]
```

Это единственный блок. Ничего больше. Фокус на одном действии.

### Состояние 2: В процессе

```
С возвращением!

[████████░░░░░░] 3 из 5 — 60%

Ты остановилась на:
Урок 3: Правильная система ежедневного ухода

[Продолжить]
```

Continue block — первое, что видишь. Не нужно ничего искать.

Ниже — список уроков:
```
✅ Урок 1: Как устроены волосы          [Пересмотреть]
✅ Урок 2: Ошибки домашнего ухода       [Пересмотреть]
▶️ Урок 3: Система ежедневного ухода   [Продолжить]
🔒 Урок 4: Домашнее восстановление     Откроется после Урока 3
🔒 Урок 5: Блонд и сложные случаи      Откроется после Урока 4
```

### Состояние 3: Курс завершён

```
Ты прошла весь курс!

Теперь у тебя есть система. Возвращайся к урокам
и бонусам когда хочешь.

[Бонусные материалы]  [Пересмотреть уроки]
```

В будущем: здесь появится AI-диагностика и upsell.

## Карточка урока

Каждый урок — карточка с чёткой визуальной иерархией:

| Элемент | Для чего |
|---------|---------|
| Номер + название | Идентификация |
| Длительность | Оценка времени |
| Статус (иконка) | Где я |
| Краткое описание | Зачем этот урок |
| Кнопка действия | Что делать |

### Визуал по состояниям

| Состояние | Фон карточки | Иконка | Кнопка |
|-----------|-------------|--------|--------|
| Completed | Лёгкий зелёный/тёплый | ✅ | "Пересмотреть" |
| In Progress | Акцентный (золотой/primary) | ▶️ | "Продолжить" |
| Unlocked | Стандартный | 📖 | "Начать" |
| Locked | Серый, приглушённый | 🔒 | — (текст: "Откроется после...") |

## Страница урока

### Структура

**Шапка:**
Урок 3 из 5 · Правильная система ежедневного ухода · ~20 мин

**Видео:**
Основной контент. Занимает центральную часть экрана.
На мобильном — full width.

**Под видео — три секции:**

**Секция 1: Что ты узнала**
- 5–7 ключевых выводов
- Не пересказ, а главные мысли
- Можно вернуться и перечитать без пересмотра видео

**Секция 2: Что сделать сегодня**
- 2–3 конкретных действия
- Пример: "Проверь температуру воды при мытье — она должна быть тёплой, не горячей"
- Практично, выполнимо, конкретно

**Секция 3: Материалы**
- Связанные PDF, чеклисты
- Ссылки на рекомендованные средства (если есть)

**Внизу:**
Кнопка "Отметить как пройденный" → разблокирует следующий урок.

## Логика Unlock

### Правила

1. Урок 1 — доступен сразу после покупки
2. Уроки 2–5 — последовательно, после завершения предыдущего
3. Завершение = кнопка "Отметить как пройденный" ИЛИ видео досмотрено до 90%
4. Бонусы — доступны сразу, параллельно урокам

### Почему последовательно

- Курс — это система, каждый урок опирается на предыдущий
- Предотвращает cherry-picking ("посмотрю только про блонд")
- Создаёт чувство прогресса и достижения
- Мотивирует завершить

## Continue Where You Left Off

### Что сохраняется

- Текущий урок (первый незавершённый)
- Позиция видео в секундах (Phase 2)
- Статусы всех уроков

### Как работает

При входе в dashboard:
```
if (есть незавершённый урок)
  → показать "Продолжить [Урок X]"
if (все завершены)
  → показать "Курс пройден" + бонусы
if (ничего не начато)
  → показать "Начать Урок 1"
```

### MVP vs Phase 2

**MVP:** Сохраняется статус урока (начат/завершён). Видео начинается сначала.
**Phase 2:** Позиция видео сохраняется каждые 30 секунд. При возвращении — видео стартует с того места.

## Бонусы

### Доступ
Доступны сразу после покупки. Не зависят от прогресса уроков.

### Содержание (MVP)
- PDF: Гайд по подбору средств
- PDF: Чеклист ежедневного ухода
- PDF: Список рекомендованных средств

### Как выглядит
```
🎁 Бонусные материалы

📄 Гайд по подбору средств         [Скачать]
📄 Чеклист ежедневного ухода       [Скачать]
📄 Рекомендованные средства        [Открыть]
```

## Профиль

### MVP
- Имя и email (из Google OAuth)
- Аватар
- Дата покупки
- Результат диагностики (если проходил квиз)
- Кнопка "Выйти"

### Чего нет в MVP
- Смена пароля (OAuth, пароля нет)
- Настройки уведомлений
- Фото-дневник волос (Phase 3)

## Будущие расширения (Phase 2–3)

| Фича | Фаза | Что даёт |
|------|------|---------|
| Resume video playback | Phase 2 | "Продолжить с 4:32" |
| Email-напоминания | Phase 2 | Retention — "ты остановилась на Уроке 3" |
| AI-диагностика | Phase 3 | Загрузи фото → получи анализ |
| Подбор средств | Phase 3 | Персональные рекомендации |
| Фото-дневник | Phase 3 | Отслеживание прогресса |
| Курс для мастеров | Phase 3 | Второй продукт в том же dashboard |
| Запись к мастеру | Phase 3 | Booking → студия или онлайн |

Архитектура dashboard готова к этим расширениям: модульная, каждый блок — отдельный компонент.

## Что такое value ladder

Лестница ценности — это не "как впихнуть больше продуктов". Это **логика отношений с пользователем**: каждая следующая ступень даёт больше ценности и требует больше доверия.

Пользователь поднимается по лестнице не потому, что его заставили, а потому что каждая предыдущая ступень доказала: "здесь мне реально помогают".

---

## Ступени

### Ступень 0: Бесплатный контент (привлечение)

**Что:** TikTok, Reels, Stories
**Ценность:** Узнавание проблемы, первое доверие к эксперту
**Стоимость:** 0€
**Цель:** Привести на сайт
**Метрика:** Views, saves, link clicks

### Ступень 1: Бесплатная диагностика (вовлечение)

**Что:** Квиз + результат + бесплатный урок
**Ценность:** Персональный результат + реально полезный контент
**Стоимость:** 0€ (email не обязателен на MVP)
**Цель:** Доказать ценность, подготовить к покупке
**Метрика:** Quiz completions, free lesson views

### Ступень 2: Основной курс (монетизация)

**Что:** "Домашняя система восстановления волос" — 5 уроков
**Ценность:** Полная система: от диагностики до восстановления
**Стоимость:** 49€ (специальная) / 89€ (базовая)
**Цель:** Основной revenue, решение проблемы пользователя
**Метрика:** Sales, completion rate, satisfaction

### Ступень 3: Premium (углубление) — Phase 2–3

**Что:** AI-диагностика по фото, персональный подбор средств
**Ценность:** Индивидуальные рекомендации
**Стоимость:** Включена в курс (увеличивает ценность) или отдельная подписка
**Цель:** Увеличить perceived value, retention

### Ступень 4: High-touch (личный контакт) — Phase 3

**Что:** Онлайн-консультация с Еленой
**Ценность:** 1-на-1 анализ, персональный план
**Стоимость:** 50–100€
**Цель:** Premium revenue, воронка в офлайн

### Ступень 5: Офлайн (максимальная ценность) — существует уже

**Что:** Визит в студию Keratin Madrid
**Ценность:** Профессиональная процедура
**Стоимость:** 120€+ за процедуру
**Цель:** Офлайн-бизнес Елены

### Ступень 6: Pro (новая аудитория) — Phase 3

**Что:** Курс для мастеров
**Ценность:** Профессиональное обучение
**Стоимость:** 199–299€
**Цель:** Второй поток revenue, новая аудитория

---

## Каждая ступень доказывает ценность следующей

```
Reels: "Она знает, о чём говорит"
  → Квиз: "Это реально про мои волосы"
    → Бесплатный урок: "Это полезно, хочу больше"
      → Курс: "Теперь у меня система"
        → AI: "Теперь я знаю, что подходит именно мне"
          → Консультация: "Хочу индивидуальный план"
            → Студия: "Хочу профессиональную процедуру"
```

Переход между ступенями — **естественный, не агрессивный**. Никогда не давим. Показываем следующую ступень только когда текущая дала результат.

## MVP монетизация

На MVP — **одна точка монетизации**: курс за 49€.

Не нужно:
- Подписок
- Нескольких продуктов
- Upsell в checkout
- Промокодов
- Реферальной системы

Одна цена. Одна кнопка. Одна транзакция. Stripe Checkout.

Всё остальное — Phase 2+.

---

## Юнит-экономика (ориентир)

| Параметр | Значение |
|---------|---------|
| Цена курса | 49€ |
| Stripe комиссия | ~1.7€ (3.4%) |
| Чистый доход с продажи | ~47€ |
| CAC (если через органику) | ~0€ |
| CAC (если через рекламу) | 5–15€ (целевой) |
| Конверсия visitor → quiz | 30–50% (ожидание) |
| Конверсия quiz → free lesson | 60–80% |
| Конверсия free lesson → purchase | 5–15% |
| Break-even при рекламе | ~1 продажа на 3–10 посетителей flow |

**Ключевое:** На старте весь трафик — органический (Reels/TikTok). CAC ≈ 0. Каждая продажа — почти чистая прибыль.

## Pricing — обоснование

### Почему 49€ — правильная цена

- **Доступная.** Для девушки, которая не может тратить 120€ на салонную процедуру, 49€ — это реальная возможность получить профессиональную систему. Это не "дёшево". Это справедливо.
- **Объективно выгодная.** Курс из 3–4 салонных процедур — 480€. Этот курс учит делать то же самое дома. Экономия 430€+.
- **Ниже барьера.** 49€ — ниже "психологического потолка 50€" для онлайн-покупки. Не нужно "думать неделю".
- **Достаточно значимая.** Чтобы ощущалась как ценность, а не как "бесплатный мусор". Люди ценят то, за что заплатили.
- **Доступна из любой точки мира.** Девушка в Мадриде и девушка в Красноярске платят одинаково. Нет географического барьера.

### Контекст цены для пользователя

На сайте полезно показать не абстрактную "скидку", а реальное сравнение:

| Что | Стоимость |
|-----|----------|
| Маска, которая не подходит | 35–50€ (впустую) |
| Один визит к мастеру | 120€ |
| Курс салонного восстановления | 480€ |
| **Этот курс** | **49€ один раз, навсегда** |

Человек должен увидеть: 49€ — это меньше, чем она тратит на одну неподходящую баночку. А получает — систему на годы.

### Зачёркнутая цена 89€

Не как манипуляция "было 89, стало 49!". А как реальная будущая цена после специального предложения. Когда продукт обрастёт AI и новыми фичами — цена вырастет.

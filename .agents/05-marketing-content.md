## Что ты делаешь

### 1. Сценарии Reels / TikTok

Каждый Reel:
- Хук 0-2 сек (визуальный или текстовый)
- Тело 20-50 сек (объяснение / разбор / до-после)
- CTA последние 3-5 сек
- Длительность: 30-60 сек, максимум 90
- Субтитры обязательно (80% смотрят без звука)
- Кодовое слово: ДИАГНОСТИКА

Три типа Reels:
- Узнавание (40%): "5 признаков...", "3 причины..."
- Доверие (30%): "Мастер оценивает...", "TikTok vs мастер"
- Конверсия (30%): "До/после с объяснением", "Система vs хаос"

### 2. Email-цепочки

MVP: 2 письма
- Welcome (сразу после покупки)
- Reminder (через 3 дня если не начала)

Тон: тёплый, короткий, конкретный. Каждое письмо: 1 действие.

### 3. SEO

Мета-теги для публичных страниц:
- / (landing): title, description, og:image
- /free-lesson: title, description
- /about: title, description

### 4. Тексты для соцсетей

Bio Instagram, описания для шапки профиля, тексты для Stories.

## Правила текстов

### Абсолютные запреты
- Длинное тире (—) НИКОГДА
- "В современном мире..."
- "Важно понимать, что..."
- "Уникальная авторская методика"
- "Данный курс предоставляет возможность..."
- "Секрет красивых волос"
- "Только сегодня! Осталось 3 места!"
- Emoji через слово
- Перечисления через точку с запятой на полстраницы
- Одинаковая структура каждого абзаца

### Тест качества
Прочитай текст вслух. Если звучит как робот, рекламный ролик или лекция, перепиши. Должно звучать как живой человек, который разбирается и хочет помочь.

### Тон
- Обращение на "ты"
- Короткие предложения рядом с длинными
- Конкретика: "маска держится 10 минут, не 2"
- Разговорные обороты: "вот в чём дело", "и это нормально"
- Никогда не хейтим бренды или AI. Объясняем контекст

### Формулировки для позиционирования

**vs ChatGPT:**
Не "ChatGPT глупый". А "Ты уже спрашивала AI про волосы. Получила общий ответ. Вот почему общий не работает для твоих конкретных волос."

**vs Kérastase:**
Не "Kérastase не работает". А "Kérastase делает хорошие средства. Но без системы даже хорошее средство не даёт результат."

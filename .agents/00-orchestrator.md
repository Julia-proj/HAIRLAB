# Агент 0: Orchestrator (Supervisora)

## Кто это
Это ты. Не AI. Ты координируешь всех агентов, принимаешь финальные решения, контролируешь качество.

## Твои обязанности
1. Распределять задачи между агентами
2. Принимать решения при конфликтах между рекомендациями агентов
3. Контролировать соответствие SOURCE_OF_TRUTH.md
4. Следить за сроками (roadmap из SOURCE_OF_TRUTH раздел 12)
5. Коммитить код и деплоить

## Правила принятия решений (порядок приоритетов)
1. Пользовательская ценность (что лучше для Елены и клиенток)
2. Качество портфолио (что выглядит senior-level)
3. Архитектурная чистота (что не сломается)
4. Скорость реализации (что можно сделать в срок)
5. Стоимость (деньги на инфру, API)

## Когда вызывать какого агента

| Ситуация | Агент |
|---|---|
| Нужно решить, как устроить экран/flow | Agent 1 (Product/UX) |
| Нужно написать код фичи | Agent 2 (Developer) через Claude Code |
| Нужно спроектировать AI-промпт или pipeline | Agent 3 (AI Integration) |
| Нужно настроить аналитику или посмотреть воронку | Agent 4 (Data/Analytics) |
| Нужен текст для Reels, email, SEO | Agent 5 (Marketing) |
| Нужно проверить код перед деплоем | Agent 6 (QA/Security) |
| Нужен промпт для Lovable | Agent 1 (Product/UX) + DESIGN_SYSTEM.md |

## Модель
Ты не AI. Ты человек. Используй свой мозг + интуицию + здравый смысл.
